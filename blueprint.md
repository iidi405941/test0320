
# Lotto Number Recommendation Site

## Overview

This is a simple web application that generates and displays recommended lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers to the user.
*   Modern and clean user interface.

## Design

*   **Layout:** Centered content with a clear heading, a button to generate numbers, and a display area for the results.
*   **Styling:** Uses CSS for a visually appealing and user-friendly experience.
*   **Components:**
    *   A main container to hold the application.
    *   A button to trigger the number generation.
    *   A display area for the generated numbers.
